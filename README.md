# NuclearCraft-Neoteric
This is a re-creation of old but gold NuclearCraft on a modern MC version

- [x] Processors
- [x] Basic Recipes (Furnace, Crafting Table, etc.)
- [x] Parts and items
- [x] Fluids
- [x] Fission Fuel
- [x] Solar Panels
- [x] Processors Sided Content handling
- [x] Fission Reactor
- [x] Radiation
